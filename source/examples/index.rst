Remote Objects

Beginner's example
__________________

These examples are for absolute beginners into the world of data-acquisition

.. toctree::
    server/spectrometer/index

The code is hosted at the repository `hololinked-examples <https://github.com/VigneshVSV/hololinked-examples>`_. 
Consider also installing

* a JSON preview tool for your browser like `Chrome JSON Viewer <https://chrome.google.com/webstore/detail/json-viewer/gbmdgpbipfallnflgajpaliibnhdgobh>`_.
* `hololinked-portal <https://github.com/VigneshVSV/hololinked-portal>`_ to have an web-interface to interact with RemoteObjects (after you can run your example object)
* `hoppscotch <https://hoppscotch.io/>`_ or `postman <https://www.postman.com/downloads/>`_
